🔄 DATA FLOW EXAMPLES
═══════════════════════════════════════════════════════════════════════════════

User asks: "JARVIS, quanto manca ai 110?"
   → detect_finance_check() detects keyword
   → check_gap() loads finances.json
   → Calculates gap = target - entrate
   → Generates TTS message
   → speak() synthesizes + plays audio
   → Output: "Signore, mancano 34.50 euro..."

User asks: "apri chrome"
   → detect_tools() matches keyword
   → open_app("chrome") launches executable
   → Immediate response: "Chrome aperto Sir."
   → Zero latency (no AI call needed)

User asks: "Dimmi di matematica"
   → detect_tools() finds university keyword
   → search_documents() queries ChromaDB
   → smart_answer() calls smart_answer()
   → Ollama generates response using documents
   → speak() outputs via TTS
   → Memory saved to history

PYGAME MIXER

   ✓ Simple channel management
   
SINGLETON PATTERN FOR CONFIG

   ✓ Lazy initialization
   ✓ Thread-safe (Python GIL)
   ✓ Easy dependency injection
